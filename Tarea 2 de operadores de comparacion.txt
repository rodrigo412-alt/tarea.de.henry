buscando. por ejemplo...for(let i=0; i<10; i++){}; ahi estoy diciendo que i es igual a 0 y estonces la variable i se va a 
repetir hasta 10 veces. tambien se pueden evaluar infinitamente pero no es recomendable hacerlo por la razon que se puede

&&= esto se utiliza para agregar otra comparacion por ejemplo: if(10===10 && 9<10) ahi estoy haciendo dos comparaciones dentro
de un if

se usa un if y se ponen las || el if necesitara de un solo true/verdadero para que funcione, en el caso de las && si una es
false y la otra es verdadero no se ejecutara la funcion if
